# MNIST_GAN

极简全连接 GAN 生成 MNIST 手写数字。

## 架构

| 网络 | 结构 |
|------|------|
| Generator | 100 → 128 → 256 → 512 → 784 (28×28) |
| Discriminator | 784 → 256 → 128 → 1 (Sigmoid) |

## 训练

- 优化器：Adam (lr=0.0002)
- 批次大小：64
- 纳什均衡检测：损失 5 轮内标准差 < 0.05 自动停止

## 运行

```bash
python train.py
```

输出：`samples/` 目录下每 5 轮保存的生成样本 + 最终 16 张图。
