# 神经网络一维热传导 (PINN)

物理信息神经网络求解一维热传导方程。

## 方法

将 PDE 残差嵌入损失函数，通过自动微分计算导数：

L = L_PDE + λ_IC·L_IC + λ_BC·L_BC

## 网络架构

2 输入 (x,t) → 64 → 64 → 64 → 64 (Tanh) → 1 输出 (T)

## 训练配置

| 参数 | 值 |
|------|-----|
| 优化器 | Adam (lr=0.002, StepLR γ=0.6/3000步) |
| 训练轮数 | 15000 |
| 配点 | 5000 PDE + 500 IC + 500 BC |

## 运行

```bash
python heat_nn_solver.py
```

输出：PINN 解图 + 与有限差分对比图 + 误差统计表。
