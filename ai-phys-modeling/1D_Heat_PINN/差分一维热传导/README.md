# 差分一维热传导

显式 FTCS 有限差分法求解一维热传导方程。

## 方法

离散格式：T_i^{n+1} = T_i^n + r(T_{i+1}^n - 2T_i^n + T_{i-1}^n) + Δt·f(x_i, t_n)

r = αΔt/Δx² = 0.2401 < 0.5，满足 CFL 稳定性条件。

## 参数

| 参数 | 值 |
|------|-----|
| Δx | 0.0204 m (nx=50) |
| Δt | 0.01 s (nt=101) |
| 精度 | 一阶时间，二阶空间 |

## 运行

```bash
python heat_conduction_1d.py
```

输出：温度分布图（3D 表面、等高线、剖面、演化曲线）+ 数据表。
