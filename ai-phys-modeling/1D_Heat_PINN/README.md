# 1D_Heat_PINN

一维热传导方程数值求解，对比经典方法与深度学习。

## 控制方程

∂T/∂t = α ∂²T/∂x² + f(x,t)

α = 0.01 m²/s, x ∈ [0,1] m, t ∈ [0,1] s, f(x,t) = 5·exp(-20(x-0.5)²)
