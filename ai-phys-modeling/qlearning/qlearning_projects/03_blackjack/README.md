# 03_blackjack

概率与风险决策：21 点 Q-Learning。

## 环境

- 状态：(玩家点数, 庄家明牌, 是否有可用 A)
- 动作：0=停牌 / 1=要牌
- 奖励：赢 +1 / 输 -1 / 平 0

## 核心概念

| 概念 | 说明 |
|------|------|
| 概率决策 | 牌面随机，智能体学习最优策略 |
| 风险管理 | 要牌可能爆牌，停牌可能输给庄家 |
| 状态聚合 | ~280 个离散状态 |

## 运行

```bash
python train.py   # 训练（100000 局）
python play.py    # 与训练好的智能体对局
```

## 预期结果

胜率约 42-43%（庄家天然优势，这是最优策略水平）。
