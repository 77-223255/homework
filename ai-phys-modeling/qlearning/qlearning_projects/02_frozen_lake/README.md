# 02_frozen_lake

随机环境下的 Q-Learning：冰面迷宫。

## 环境

```
S F F F    S=起点  F=冰面(安全)  H=洞  G=目标
F H F H
F F F H
H F F G
```

## 核心概念

| 概念 | 说明 |
|------|------|
| 随机转移 | 冰面打滑，动作只有 33% 概率生效 |
| 稀疏奖励 | 只有到达目标/掉入洞时才有奖励 |
| 状态空间 | 16 个状态 |

## 运行

```bash
python train.py   # 训练（10000 局）
python play.py    # 观看智能体行动
```

## 预期结果

成功率约 70-75%（随机性限制）。
