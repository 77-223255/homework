# 04_dqn_intro

深度 Q 网络 (DQN) 入门：用神经网络替代 Q-Table。

## 问题

CartPole 环境的状态是连续的（位置、速度、角度、角速度），Q-Table 无法处理无限状态空间。

## 解决方案

用神经网络近似 Q 值函数：

```
State (4维) → [24 → 24] (ReLU) → Q-values (2维)
```

## 核心创新

| 机制 | 作用 |
|------|------|
| 经验回放 | 随机采样历史经验，打破数据相关性 |
| 目标网络 | 延迟更新，稳定训练 |

## 运行

```bash
python train.py
```
