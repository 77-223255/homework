# llmbuild

学习《Build a Large Language Model (From Scratch)》的代码实现。

## 目录结构

| 目录 | 内容 |
|------|------|
| [`1_input_data_process`](./1_input_data_process/) | 数据预处理：文本分词与 Token ID 转换 |
| [`2_attention_machasisms`](./2_attention_machasisms/) | 注意力机制：Self-Attention 实现 |

## 环境

```bash
conda activate llmbuild
```

依赖：Python 3.10, tiktoken, torch
